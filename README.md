# Sobre o Projeto
Exemplo simples de CRUD API REST

Create -> POST /tasks| Body: { "description": "string", "done": boolean }

Read -> GET /tasks

Update -> PUT /tasks/id | Body: { "description": "string", "done": boolean }

Delete -> DELETE /tasks/id
